# Rubik's Cube Articulator and Solver

## Description
A motorized Rubik’s Cube articulator robot designed to rotate each face with precision using a handheld controller. The system allows for manual face turns, scrambling the cube with a button, and solving the cube using a reverse scramble algorithm. The project emphasizes mechanical accuracy, motor control, and movement sequencing rather than a complete autonomous solving algorithm.

## Tech Stack

### Hardware
- Arduino Uno  
- 6 VEX V5 motors (one per face)  
- Speed Rubik’s Cube (for more smooth movements) 
- Handheld vex controller  
- Push buttons (scramble / solve)  
- Custom CAD-designed mounts
- VEX brain
- Axles, wiring, fasteners, screws, push buttons, battery  

### Software
- Vex V5 IDE (Python)
- CAD software

## Controls
- Bottom face: Right side top button rotates clockwise | Right side bottom button rotates counterclockwise
- Top face: Left side top button rotates clockwise | Left side bottom button rotates counterclockwise
- Right face: Right side right button rotates counterclockwise | Right side left button rotates clockwise
- Left face: Left side left button rotates counterclockwise | Left side right button rotates clockwise
- Front face: Right side top bumper rotates counterclockwise | Right side bottom bumper rotates clockwise
- Back face: Left side top bumper rotates counterclockwise | Left side bottom bumper rotates clockwise
